\doxysubsection{Detailed Description}
An example of a general-\/purpose implementation that includes the optional functionality of allowing multiple base directories for a single namespace prefix.

Given a foo-\/bar package of classes in the file system at the following paths ... \begin{DoxyVerb}/path/to/packages/foo-bar/
    src/
        Baz.php             # Foo\Bar\Baz
        Qux/
            Quux.php        # Foo\Bar\Qux\Quux
    tests/
        BazTest.php         # Foo\Bar\BazTest
        Qux/
            QuuxTest.php    # Foo\Bar\Qux\QuuxTest
\end{DoxyVerb}
 ... add the path to the class files for the \textbackslash{}\+Foo\textbackslash{}\+Bar\textbackslash{} namespace prefix as follows\+: \begin{DoxyVerb} <?php
 // instantiate the loader
 $loader = new \Example\Psr4AutoloaderClass;

 // register the autoloader
 $loader->register();

 // register the base directories for the namespace prefix
 $loader->addNamespace('Foo\Bar', '/path/to/packages/foo-bar/src');
 $loader->addNamespace('Foo\Bar', '/path/to/packages/foo-bar/tests');
\end{DoxyVerb}
 The following line would cause the autoloader to attempt to load the \textbackslash{}\+Foo\textbackslash{}\+Bar\textbackslash{}\+Qux\textbackslash{}\+Quux class from /path/to/packages/foo-\/bar/src/\+Qux/\+Quux.php\+: \begin{DoxyVerb} <?php
 new \Foo\Bar\Qux\Quux;
\end{DoxyVerb}
 The following line would cause the autoloader to attempt to load the \textbackslash{}\+Foo\textbackslash{}\+Bar\textbackslash{}\+Qux\textbackslash{}\+Quux\+Test class from /path/to/packages/foo-\/bar/tests/\+Qux/\+Quux\+Test.php\+: \begin{DoxyVerb} <?php
 new \Foo\Bar\Qux\QuuxTest;
\end{DoxyVerb}
